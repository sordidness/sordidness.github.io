JavaScript中的变量声明

发表于 2016-03-01

如何在JavaScript中声明一个变量？

看到问题相信大家都蜜汁一笑……。

我们在学校编程的时候，无论哪种语言，最基本的一课就是声明变量，可见声明变量是编程绝对不可或缺的一门功课。

在es6之前，我们都是使用var关键字来声明变量，es6中又新增了两种声明变量的关键字：let和const，我们以此来看下它们的用法。

let

let关键字是es6语法新加的，let是有块级作用域概念的，let声明的变量只能在块级单位内使用，让变量在一个较小的范围内运行，这样就减小了错误产生的可能性。

这时候会跑出一个错误，变量不存在。

这样在块级单位内部使用变量，就可以正常运行了。

const

const是一个比较特殊的关键字，const声明的变量不可修改它的值，只能在声明的时候赋值给它，否则会报错，并且const也是有块级作用域概念的。

但是，当const声明的变量是Object或Array时，可以理解为在内存中申请了一块地址，并由指针指向它，当我们在修改Object或Array的内容时，只是修改了里面的的东西，并没有更改它的地址，所以是不会报错的。

结论

var 在一定程度上存在缺陷，尽量使用let替代，不过要注意作用域的问题，当需要使用常量时，尽量使用const，可以最大程度避免产生不必要的错误，浪费开发时间。
